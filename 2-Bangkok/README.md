提交截止时间根据赛区日程而定，截止时间后不可再提交或修改比赛成果。

比赛需交付的内容包括：

- **作品代码链接（必选，原代码存放个人GitHub仓库，不用直接提交）**

- **其他任何额外的支持材料；**
